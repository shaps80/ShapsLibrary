fetchObjectWithIdentifier:entityName:autoCreate:

Returns an NSManagedObject instance from the specified entity with the given identifier.

- (id)fetchObjectWithIdentifier:(id)identifier entityName:(NSString *)entityName autoCreate:(BOOL)autoCreate

Parameters

identifier

The identifier of the object.

entityName

The name of the entity to fetch.

autoCreate

If yes, when the object is not found, it is automatically created.

Return Value

An NSManagedObject instance if found, nil otherwise.
